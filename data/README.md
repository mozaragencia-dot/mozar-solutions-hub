# Carpeta de datos (respaldo)

Aquí puedes guardar manualmente los archivos de respaldo exportados desde el sistema (`respaldo-tacam-YYYY-MM-DD.json`).

## Flujo recomendado
1. En la app abre **Reportes y Gráficos**.
2. Usa **Descargar respaldo (.json)**.
3. Mueve ese archivo a esta carpeta `data/` para tener historial dentro del proyecto.

> Nota: por seguridad, revisa que los respaldos no contengan datos sensibles antes de compartir el repositorio.
