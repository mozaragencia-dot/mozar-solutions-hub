# Pulse 50/50

Webapp estática y responsive pensada como demo rápida para Android y iPhone.

## Qué incluye

- Pantalla dividida 6/6 con porcentaje visible para cada lado.
- Pregunta política con dos opciones: Sí / No.
- Campo para ingresar RUT antes de votar.
- Limpieza inmediata del RUT desde la interfaz después del envío.
- Balance visual suavizado para que el marcador se mantenga cerca de 50% / 50%.
- Colores que se intensifican según el lado que va ganando.

## Cómo abrir

Abre `index.html` directamente en un navegador móvil o de escritorio.
